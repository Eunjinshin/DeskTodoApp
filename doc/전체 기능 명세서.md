# WeGet TODO — 전체 기능 명세서

> 보안 강화 Electron 기반 데스크톱 TODO 애플리케이션

---

## 프로젝트 구조

```
02.weget/
├── backend/            # Main Process
│   ├── index.js        # 엔트리포인트, CSP, 보안 이벤트
│   ├── window.js       # BrowserWindow (프레임리스)
│   ├── ipc-handlers.js # IPC 핸들러 (TODO/Schedule/Window)
│   ├── db/             # SQLCipher DB
│   ├── security/       # key-mgr, validator, integrity
│   └── network/        # axios-sec (SSL Pinning)
├── preload/            # Context Bridge (화이트리스트 IPC)
├── frontend/           # React + Vite + Tailwind CSS v3
│   └── src/
│       ├── api/        # 보안 IPC 래퍼
│       ├── components/ # UI 컴포넌트
│       ├── hooks/      # 커스텀 Hook
│       ├── context/    # 전역 상태
│       └── utils/      # XSS 방어 유틸리티
└── doc/                # 문서
```

## 기술 스택

| 항목 | 기술 |
|------|------|
| Runtime | Electron v28+ |
| Backend | Node.js v18+ |
| Database | SQLite + SQLCipher (`@journeyapps/sqlcipher`) |

## 기능 목록

### 1. TODO 관리 (메인 기능)

| 기능 | 설명 |
|------|------|
| TODO 등록 | 제목, 내용, 우선순위, 마감일, 태그 입력 |
| TODO 완료 | 체크 시 취소선 + 완료 시간(`HH:mm`) 자동 기록 |
| TODO 삭제 | 호버 시 삭제 버튼 표시 |
| 우선순위 | 상(🔴)/중(🟡)/하(⚪) — 상이 목록 상단으로 자동 정렬 |
| 태그 | `#태그` 형태로 TODO에 태그 등록 및 표시 |
| 날짜 기본값 | 마감일 기본값이 오늘로 설정 |
| 날짜별 필터 | 폼 아래 날짜 선택기로 해당 날짜 TODO만 표시 (기본 오늘, 전체 보기 가능) |

#### 우선순위 색상

| 등급 | 배경색 | 글자색 |
|------|--------|--------|
| 상 | `bg-rose-500/20` (부드러운 빨강) | `text-rose-300` |
| 중 | `bg-amber-500/20` (부드러운 노랑) | `text-amber-300` |
| 하 | `bg-slate-500/20` (회색) | `text-slate-400` |

## 보안 기능

### DB 암호화
- SQLCipher 단일 레이어 (DB 파일 전체 암호화)
- 마스터 키: Electron `safeStorage` API (DPAPI/Keychain)

### IPC 보안
- Context Isolation + `nodeIntegration: false`
- Preload 화이트리스트 기반 채널 필터링 (15개 채널)
- `event` 객체 Renderer 노출 차단

### XSS 방어
- React JSX 자동 이스케이프
- `sanitize.js` — HTML 특수 문자 이스케이프 + `Object.freeze()`
- `dangerouslySetInnerHTML` 사용 금지

### 네트워크 보안
- CSP(Content Security Policy) 개발/프로덕션 분기
- SSL Pinning + TLS 1.2+ 강제 (Axios)
- 외부 URL 네비게이션 완전 차단

### 기타
- 파일 무결성 HMAC-SHA256 검증
- 입력값 검증 (타입/길이/SQL Injection 패턴)
- 감사 로그 (audit_log 테이블)
- 단일 인스턴스 잠금 (중복 실행 방지)
- 프로덕션 DevTools 비활성화

## DB 스키마

### todos
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | TEXT PK | UUID v4 |
| title | TEXT | 제목 |
| content | TEXT | 내용 (선택) |
| tag | TEXT | 태그 (선택) |
| is_done | INTEGER | 완료 여부 (0/1) |
| priority | INTEGER | 우선순위 (0=하, 1=중, 2=상) |
| due_date | TEXT | 마감일 (YYYY-MM-DD) |
| completed_at | TEXT | 완료 시간 (ISO 8601) |
| created_at | TEXT | 생성일 |
| updated_at | TEXT | 수정일 |

### schedules
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | TEXT PK | UUID v4 |
| title | TEXT | 일정 제목 |
| content | TEXT | 내용 (선택) |
| start_date | TEXT | 시작일 |
| end_date | TEXT | 종료일 (선택) |
| recurrence | TEXT | 반복 (once/daily/weekly/monthly/yearly) |
| color | TEXT | 색상 코드 |
| created_at | TEXT | 생성일 |
| updated_at | TEXT | 수정일 |

### audit_log
| 컬럼 | 타입 | 설명 |
|------|------|------|
| id | INTEGER PK | 자동 증가 |
| action | TEXT | CREATE/UPDATE/DELETE |
| target_id | TEXT | 대상 ID |
| target_type | TEXT | todo/schedule |
| timestamp | TEXT | 시간 |
| detail | TEXT | 변경 상세 |

## IPC 채널 목록

| 채널 | 방향 | 설명 |
|------|------|------|
| `todo:create` | Renderer → Main | TODO 생성 |
| `todo:read` | Renderer → Main | TODO 단건 조회 |
| `todo:update` | Renderer → Main | TODO 수정 |
| `todo:delete` | Renderer → Main | TODO 삭제 |
| `todo:list` | Renderer → Main | TODO 목록 |
| `schedule:create` | Renderer → Main | 일정 생성 |
| `schedule:list` | Renderer → Main | 일정 목록 |
| `schedule:update` | Renderer → Main | 일정 수정 |
| `schedule:delete` | Renderer → Main | 일정 삭제 |
| `window:pin` | Renderer → Main | 항상 앞에 |
| `window:unpin` | Renderer → Main | 핀 해제 |
| `window:minimize` | Renderer → Main | 최소화 |
| `window:close` | Renderer → Main | 닫기 |
| `window:set-opacity` | Renderer → Main | 투명도 변경 |
| `todo:sync-status` | Main → Renderer | 동기화 상태 |
