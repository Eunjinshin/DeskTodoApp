## 🐛 발생한 문제 (Issue Description)
화면 고정(Always on Top) 기능과 관련하여 기능 연동 및 UI 표시에 문제가 발생하고 있습니다.

### 1. 기능 연동 및 피드백 누락 이슈
- 상단의 **핀(Pin) 아이콘**을 클릭했을 때 화면 고정 기능이 실제로 동작하는지 확신할 수 없습니다.
- 상단 핀 아이콘을 클릭하여 활성화하더라도, 설정 창의 **[화면 고정]** 옵션이 `ON`으로 동기화되지 않는 연동 문제가 있습니다.

### 2. UI 표시 이슈
- 상단 핀 아이콘을 클릭했을 때, 활성화되었음을 나타내는 시각적인 피드백(색상 변경, active 스타일 적용 등)이 없어 사용자가 상태를 인지하기 어렵습니다.

<br/>

## 🎯 기대 결과 (Expected Behavior)
- 핀 아이콘 클릭 시 앱이 최상단에 고정되어야 하며, 이 상태가 환경설정의 '화면 고정' 옵션과 양방향으로 동기화되어야 합니다. (어느 쪽에서 변경하든 상태가 일치해야 함)
- 핀 아이콘이 활성화되었을 때 아이콘의 색상이 바뀌거나 배경이 추가되는 등 눌러진 상태가 직관적으로 보여야 합니다.
