# 기능 보완 Implementation Plan

## 코드 리뷰(가독성/성능/보안) 결과 + 신규 기능 구현 계획

### 리뷰 반영 사항
- 🔴 보안: Prepared Statement로 전환, 이벤트 이중 등록 제거
- 🟡 성능: Promise 래퍼 통합, 감사 로그 비동기 분리
- 🟢 가독성: IPC wrapHandler 패턴, queries.js 래퍼 통합

### 신규 기능
1. **탭 UI** — TODO / 달력 / 설정 전환
2. **완료 시간** — 체크 시 completed_at 기록 및 표시
3. **달력 일정** — react-calendar + 색상 선택 + CRUD
4. **핀(Always-on-Top)** — 타이틀바/설정에서 토글
5. **투명도** — 사용자 설정 슬라이더 (30~100%)
6. **프레임리스 창** — 커스텀 타이틀바 (드래그/핀/닫기)

### 보안 전략
- 달력 일정: XSS 방어용 sanitize 동일 적용
- 창 제어 IPC: Preload 화이트리스트 추가
- 프레임리스: WebkitAppRegion CSS만 허용
